Kode Status & Pola Error

200 OKPermintaan berhasil, response seperti contoh di atas.
201 CreatedBerhasil membuat resource (tergantung implementasi, bisa juga 200).
400 / 422Validasi gagal / Bad Request. Biasanya bentuk: { "status": false, "message": "...", "errors": { field:["msg"] } }
404 Not FoundResource tidak ditemukan.
500 Server ErrorKesalahan di sisi server.
