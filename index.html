Krusit API — Dokumentasi Penggunaan

Base URL: https://krusit.up.railway.app
⬇️ Download JSON (Postman)

Quickstart

  1. Semua request gunakan header: Accept: application/json dan Content-Type: application/json.
  2. Gunakan URL lengkap per method di bawah (ada tombol Copy untuk URL & Body).
  3. Import koleksi Postman lewat tombol Download JSON jika diperlukan.
Login Makanan: List Makanan: Create Makanan: Update Makanan: Delete Minuman: List Minuman: Create Minuman: Update Minuman: Delete

POST Login

URL (copy)
Headers
Accept: application/json, Content-Type: application/json
Body (copy)
{
  "email": "rezkynurintan13@gmail.com",
  "password": "12345678"
}
Response 200 (contoh) 
{
  "status": "success",
  "user": {
    "id": 1,
    "name": "Admin Krusit 1",
    "email": "rezkynurintan13@gmail.com",
    "email_verified_at": null,
    "created_at": "2025-10-08T13:54:43.000000Z",
    "updated_at": "2025-10-08T13:54:43.000000Z",
    "role": "admin"
  }
}
Login saat ini belum mengembalikan token. Endpoint lain diasumsikan public.
